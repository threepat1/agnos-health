แชทบอทฟอรัมสุขภาพ Agnos
แชทบอทนี้ช่วยตอบคำถามเกี่ยวกับสุขภาพโดยใช้โพสต์จริงจากฟอรัมสุขภาพชื่อ agnoshealth.com

มันทำงานยังไง
คุณถามคำถามเกี่ยวกับสุขภาพ
แชทบอทจะค้นหาโพสต์ที่ใกล้เคียงจากฟอรัม
แล้วตอบกลับโดยใช้โมเดล AI ที่รันอยู่ในเครื่องคุณเอง
ข้อมูลของคุณจะปลอดภัยเพราะทุกอย่างทำงานในเครื่องคุณ

วิธีทำงาน
เก็บข้อมูล: ดึงโพสต์สุขภาพจากฟอรัม
แปลงเป็นเวกเตอร์: แปลงโพสต์เป็นตัวเลขเพื่อให้คอมค้นหาเร็วขึ้น
ค้นหา: เมื่อคุณถาม แชทบอทจะหาโพสต์ที่เกี่ยวข้องที่สุด
ตอบคำถาม: ใช้ AI ในเครื่อง (ชื่อ Ollama) ตอบคำถามจากโพสต์ที่หาเจอ

วิธีติดตั้งและใช้งาน
1. ติดตั้ง Ollama
ต้องใช้ Ollama เพื่อรัน AI ในเครื่อง

ดาวน์โหลดที่นี่: https://ollama.ai/

เปิดเทอร์มินอลแล้วพิมพ์:

```bash
ollama pull llama3
```


2. ดาวน์โหลดโค้ด
โหลดโปรเจกต์นี้ลงเครื่อง:

```bash
git clone <repository-url>
cd <repository-folder>
```

3. สร้างสภาพแวดล้อม Python
สร้าง virtual environment เพื่อจัดการแพ็กเกจ:

```bash
python -m venv venv
```

source venv/bin/activate   # บน Windows ใช้ venv\Scripts\activate

4. ติดตั้งแพ็กเกจที่ต้องใช้
```bash
pip install -r requirements.txt
```

5. สร้างฐานข้อมูลเวกเตอร์
รันคำสั่งนี้เพื่อดึงโพสต์จากฟอรัมและสร้างฐานข้อมูลค้นหา:

```bash
python vector_store.py
```

6. เริ่มแชทบอท
รันคำสั่งนี้เพื่อเปิดเว็บแอปแชทบอท:

```bash
streamlit run streamlit_app.py
```

เบราว์เซอร์จะเปิดขึ้นมา เริ่มถามคำถามเกี่ยวกับสุขภาพ
